# Addition Practice Worksheet

A simple web-based tool for generating printable addition practice worksheets for elementary students.

## Features

- **Automatic Problem Generation**: Creates 20 unique addition problems each time
- **Smart Difficulty Distribution**: 80% of problems have sums greater than 10, 20% have sums of 10 or less
- **Range Control**: All problems use numbers 1-20 with sums capped at 20
- **Print-Friendly**: Designed for easy printing with clean layout and answer lines
- **Two-Column Layout**: Problems are organized in two columns for efficient use of paper
- **Duplicate Prevention**: Ensures no repeated problems (including reverse operations like 5+3 and 3+5)

## How to Use

1. Open `index.html` in a web browser
2. The page will automatically generate 20 addition problems
3. Click "Generate New Worksheet" to create a new set of problems
4. Print the page for students to complete (the button will be hidden when printing)
