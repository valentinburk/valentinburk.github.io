# Subtraction Practice Worksheet

A simple web-based tool for generating printable subtraction practice worksheets for elementary students.

## Features

- **Automatic Problem Generation**: Creates 20 unique subtraction problems each time
- **Smart Difficulty Distribution**: 90% of problems are more challenging (larger numbers), 10% are simpler
- **Range Control**: All problems subtract from numbers 1-20 with results that are never negative (minimum 0)
- **Print-Friendly**: Designed for easy printing with clean layout and answer lines
- **Two-Column Layout**: Problems are organized in two columns for efficient use of paper
- **Duplicate Prevention**: Ensures no repeated problems

## How to Use

1. Open `index.html` in a web browser
2. The page will automatically generate 20 subtraction problems
3. Click "Generate New Worksheet" to create a new set of problems
4. Print the page for students to complete (the button will be hidden when printing)
